# Project Summary
**PipingBox** is a web application designed to streamline the job application process for job seekers and employers within the industrial sector. It provides a secure platform for managing professional documents and utilizes Supabase for backend services. Key features include a **Professional Referral & Rewards System** for networking, a **Worker Search Module** for accessing real-time worker profiles, and an **Enhanced Notifications System** for timely updates. The project aims to enhance user experience, promote professional growth, and facilitate efficient job placements.

# Project Module Description
The main functional modules include:
- **Worker Documents Module**: Manages professional documents with categorization and visibility control.
- **Certifications Module**: Handles the creation and management of certifications with row-level security and improved error handling.
- **Candidate Profile Module**: Displays candidate information with options for CV preview and download, including an **Availability & Mobility Section**.
- **Professional Referral & Rewards Module**: Generates unique referral codes and tracks referrals, featuring a **ProNetworkWidget** on the Dashboard.
- **Worker Search Module**: Connects to Supabase for real worker profiles, utilizing enhanced query logic for refined search results.
- **Invite to Job Feature**: Allows company users to send job invitations to candidates through a modal interface.
- **Pending Invitations Widget**: Displays pending job invitations for workers, including job title and action buttons.
- **Job Invitations Status Widget**: Shows counts for Pending, Accepted, and Declined invitations.
- **User Profile Management**: Automatically creates user profiles upon registration, ensuring all fields are populated and supports referral tracking.
- **Admin Dashboard**: Displays user statistics and includes a feature to backfill missing profiles for existing users.
- **Admin Registros Module**: Combines data from `auth.users` and `profiles` to show all registered users with comprehensive stats.
- **Login and Registration Enhancements**: Supports account type selection (Worker vs Company) and Google sign-in.
- **Onboarding Wizard**: Guides new users through a step-by-step profile setup process with instant saving.
- **Enhanced Notifications System**: A comprehensive notifications center for users and admins, featuring real-time updates.
- **Unified Profile Completion System**: Centralized system for calculating and displaying profile completion metrics.

# Technology Stack
- React: Frontend framework for building the user interface.

# Usage
1. **Install Dependencies**: Run the package manager to install all necessary dependencies.
   ```
   pnpm install
   ```
2. **Build the Project**: Use the build command to compile the application.
   ```
   pnpm run build
   ```
3. **Run the Application**: Start the application using the appropriate command for your environment.
