FEATURE — Worker Profile Intelligence Foundation

IMPORTANT:
Do not redesign the platform.
Do not break existing Profile/CV, Jobs, Applications, Company Candidates, Messages, Admin Center, or role system.

GOAL:
Upgrade the Worker Profile / CV system into a modular professional profile where users can manage CV, work experience, certifications, documents, and visibility.

This profile must become the professional identity system of PipingBox.

==================================================
PHASE 1 — PROFILE MODULES
=========================

Add modular sections inside Profile / CV:

1. Basic Professional Info

* full name
* title/role
* current company
* location
* years of experience
* bio
* skills

2. CV Upload

* upload PDF CV
* show uploaded file name
* replace CV
* remove CV
* privacy toggle:
  visible to companies: yes/no

3. Work Experience
   Allow user to add multiple experiences:

* job title
* company
* project/site
* country/location
* start date
* end date
* current job toggle
* description
* responsibilities
* visibility toggle

4. Certifications
   Allow user to add multiple certifications:

* certificate name
* issuing body
* certificate number
* issue date
* expiry date
* file upload
* verification status: pending / verified / rejected
* visibility toggle

5. Documents
   Allow additional document uploads:

* document type
* file
* notes
* visibility toggle
