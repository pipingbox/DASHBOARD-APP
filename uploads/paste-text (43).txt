FEATURE — Certification expiry alerts + edit/delete actions.

IMPORTANT:
Do not redesign the profile page.

GOAL:
Complete the certification management system.

==================================================
PART 1 — CERTIFICATION ACTION BUTTONS
=====================================

For each certification card, add:

* Edit
* Delete / Remove
* Renew

Edit:

* opens the certification modal with current data loaded
* allows updating fields
* saves changes to Supabase

Delete:

* asks for confirmation before deleting
* deletes only the selected certification
* shows success toast

Renew:

* opens edit modal focused on:

  * new issue date
  * new expiry date
  * updated file upload if needed

==================================================
PART 2 — EXPIRY STATUS
======================

Calculate certification status from expiry_date:

* Valid: more than 90 days remaining
* Expiring soon: 90 days or less
* Critical: 30 days or less
* Expired: expiry date passed

Show clear badges:

* Valid
* Expiring soon
* Critical
* Expired

Spanish translations:

* Válido
* Próximo a caducar
* Crítico
* Caducado
